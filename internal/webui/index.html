📖 怎么用？点开看 3 分钟上手指南（含例子）/ How to use — quick guide with examples
① 排布局——从「Segment 仓库」把想要的胶囊拖进「布局编排」的行里（最多 3 行）；行内拖动排序，拖回仓库停用。
② 调外观——「外观」卡片选主题/图标集/圆角；开「紧凑模式」全局去前缀。
③ 单颗自定义——点任意胶囊右侧的小圆点 🎨：前景色、底色、前缀、加粗随便改（任意 RGB 取色器）；右键圆点重置。 例：把 today 前景改 #ff8800、前缀填 Today ，就从「今日 $9.40」变成橙色的「Today $9.40」；前缀留空则只剩「$9.40」。
④ 加自己的东西——「自定义」卡片点「＋添加插槽」： 例 1：名字 备注 + 文本 搬砖中 + 粉色 → 状态栏出现粉色「搬砖中」； 例 2：名字 ip + 命令 curl -s ifconfig.me + 蓝色 → 显示公网 IP（1s 超时 10s 缓存，断网只是这颗消失）； 例 3：命令 node --version → 显示 Node 版本。建好后从仓库区「自定义插槽」分组拖进布局。
⑤ 保存——右上角「保存生效」（预览只是试穿，保存才落盘）；Claude Code 下次刷新状态栏即生效，无需重启。
完整文档见仓库 docs/USAGE.md；虚线胶囊 = 已启用但条件未满足的示例占位，点预览下方折叠条看原因。

外观Appearance

主题决定整套配色；图标集按终端字体能力选择；胶囊背景关闭后退化为彩色文字 + │ 分隔。
Theme sets the full palette; pick an icon set your terminal font supports; turning pills off falls back to colored text with │ separators.

布局编排Layout Editor

拖动胶囊在行间移动排序（最多 3 行）；拖到下方「仓库」卡片即停用。悬停任意胶囊可查看功能说明。
🎨 每颗胶囊右侧的小圆点＝外观自定义入口：点击可改 前景色 / 底色 / 前缀 / 加粗（任意 RGB），右键圆点一键重置；已自定义的圆点发紫光。
Drag chips between rows to reorder; drop into the pool below to disable. Click the dot on any chip to customize color / pill background / label / bold — right-click the dot to reset.

Segment 仓库Idle Segment Pool

未启用的 segment，按功能块分组。把想要的拖进上方任意一行即启用。
Disabled segments, grouped by function. Drag one into a row above to enable it.

自定义Custom Segments

给 text / cmd 两个 segment 提供内容：静态文本原样显示；命令取输出首行（1 秒超时，10 秒缓存，命令失败不影响其他胶囊）。填好后把对应胶囊拖进上方布局即可。
Content for the text / cmd segments: static text is shown as-is; the command's first output line is shown (1s timeout, 10s cache, failures never break other pills). Fill in, then drag the chip into a row above.
自定义插槽 Custom Slots——想要几个加几个：起个名，填静态文本或命令（二选一，都填时文本优先），挑个颜色， 对应胶囊会出现在上方仓库区，拖进布局即用。
Add as many slots as you like: name it, give it text or a command, pick a color — the chip shows up in the pool above.
